##Proyecto
Nuestro proyecto es un piano que tiene varias canciones almacenadas, también guarda la puntuación

##Tutorial de instalación
```
  $ git clone git@github.com:Losnen/Proyecto-DSI.git
  $ npm install
  $ bower install
  $ mongod
  $ node app.js
```

##Tecnologías usadas:
* Node JS
* Express
* EJS
* MongoDB
* Mongoose
* SASS
* Bower
* Gulp
